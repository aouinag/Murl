# MURL
**MURL** is a minimal blogging engine.

You can write your posts using markdown and drop your files into *content* dierctory and voila ! 

## The name
Markdown + URL = MURL !

## How it works
* Download the zip file
* Unzip it
* Drop files to your server 
* Edit config  ( index.php)
* Add content 
* Enjoy !
